Detailed Description

Classe principale de l'approche finale.

Cette classe gère l'action de l'approche finale ainsi que l'algorithme de l'approche finale.

std::list< std::vector< Point > > FinalApproaching::objectsConstruction ( std::vector< float >  ranges,
float  angleMin,
double  angleInc,

float  margin = 0.10 
)

Crée les objets a partir du tableau de points des données lasers.

La segmentation s'effectue sur la différence de range entre deux points voisins, si elle est supérieur à margin (5 centimètres recommandés), un nouvel objet est créé. Les points comportant un range hors des limites de rangeMin / rangeMax seront ignorés (mais pas leurs voisins, pour l'instant)

Parameters

[in]angleMinThe angle minimum
[in]angleIncThe angle increment
