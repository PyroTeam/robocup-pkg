Detailed Description

Classe abstraite pour une interface commune de modificateurs de grid map.

Les modificateurs peuvent être utiles pour, par exemple, lisser la map ou ajouter des dégradés autours des objets. Ces modificateurs peuvent être chainés pour faciliter leur utilisation.
