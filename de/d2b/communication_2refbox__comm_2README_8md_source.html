1 Communication avec la Referee Box
2 =================================
3 
4 Ce package est actuellement fonctionnel avec la branche de développement **timn/rcll-2015** de la refereeBox
5 
6 Description
7 -----------
8 Ce package implémente la communication avec la referee Box en convertissant les messages au format ROS utilisable par les autres noeuds.
9 
10 Noeuds
11 ------
12 * `refBoxComm` : noeud principal de la communication
13 
14 Launchfiles
15 -----------
16 * `refBoxComm.launch` : lance le noeud `refBoxComm` et donne la possibilité de modifier divers paramètres au lancement (nom d'équipe, couleur d'équipe, numéro et nom du robot)
17 * `refDebugddd.launch` : idem que `refBoxComm.launch`, mais en lançant le noeud en mode debug via le logiciel ddd
18 
19 Installation de la referee Box
20 ------------------------------
21 
22 La procédure d'installation est décrite sur le wiki de la refereeBox dans la section reservée à Ubuntu 14.04, [ici](https://trac.fawkesrobotics.org/wiki/LLSFRefBox/Install#InstallRefboxonUbuntu14.04trusty).
23 J'indique ici la procédure complète car plusieurs modifications sont necessaires pour que le package refbox_comm compile correctement :
24 
25 Installer les dépendances :
26 
27  sudo apt-get install libmodbus-dev protobuf-compiler libprotobuf-dev libprotoc-dev libboost-all-dev \
28  libglibmm-2.4-dev libgtkmm-3.0-dev libncursesw5-dev libyaml-cpp-dev libavahi-client-dev git \
29  libxt-dev libxaw7-dev libncurses5-dev autoconf autogen libtool libyaml-dev
30 
31 Installation de Clips 6.30 : ​http://sourceforge.net/projects/clipsmm/files/clips/6.30.0.20090722svn/clips-6.30.0.20090722svn.tar.bz2/download
32 
33 Compiler :
34
